A dica de hoje é sobre o jupyter notebook, um projeto open source, bem legal e muito utilizado na área de ciência de dados.


:caderno_com_capa_decorada: Jupyter Notebook

O que é?

É uma interface gráfica que permite a edição de notebooks em um navegador web. Mas o que são esses notebooks?

São um conjunto de células executáveis ou de conteúdo que podem ser compartilhadas. Na prática podemos escrever um texto, com fórmulas matemáticas e imagens e incluir em outra célula um código que pode ser reproduzido por outra pessoa.

Atualmente podemos utilizar o jupyter com várias linguagens como Lua, Ruby, Python, R, Haskell, F#, Perl, Javascript etc.

Como eu instalo?

Você pode instalar utilizando o comando python3 -m pip install jupyter.

Teste a instalação através do comando jupyter --version .

Como roda?

Para rodar utilize o comando jupyter notebook.

fun fact: Seu nome vem da junção das palavras Julia, Python e R, que são as linguagens originalmente suportadas pela ferramenta.
